Requirement :

1. php 8.1.28
2. mysql 8
3. composer

Based on :
Laravel 10.10

How to install :

1. clone this repo !!!
2. composer install
3. npm run prod
4. php artisan migrate
5. php artisan db:seed
6. php artisan db:seed --class=RoleSeeder
7. php artisan db:seed --class=PermissionSeeder
8. enjoy
